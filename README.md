## Setup
#### Install dependencies
```
npm install
```
#### Start Json-server on port 5000
```
npm run server
```
#### Start React app on port 3000
```
npm start
```
#### Create Prod build
```
yarn run build
```

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
